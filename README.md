<!-- What is the difference between var, let, and const  -->

var → পুরনো, সব জায়গায় কাজ করে, বাগ হতে পারে।

let → নতুন, মান পাল্টানো যায়, সবচেয়ে বেশি ব্যবহার হয়।

const → নতুন, মান পাল্টানো যায় না।
